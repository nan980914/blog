# 一、减肥之饮食计划（工作日）

# 早餐

  1. 两个鸡蛋
  2. 一粒Omega3 鱼油

# 午餐

  1. 150g肉 + 一个鸡蛋（保证蛋白质）
  2. 50g主食（粗粮为主，也包括高碳水蔬菜）
  3. 200g低碳水蔬菜 （力求好吃）

# 下午加餐🥞

  1. 午睡醒来一杯黑咖啡
  2. 下午茶 / 一个水果 / 一盒100g的酸奶

# 晚餐

  1. 点外卖里的健康餐（一定要吃碳水）
  2. 如果晚上去健身房锻炼了，锻炼完回家喝15g蛋白粉

# 补充

白天一定要多喝水！
