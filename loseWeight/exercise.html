# 一、减肥之运动计划（工作日）

总计划：空腹有氧 + 团课 / 晚上回家：( keep / b站 )

# 空腹有氧

在身体能承受的了的情况下尽量做到空腹有氧（其实最重要的是早上起得来😅 ），初期先设定在20min，有利于坚持下去。

具体方式可以随心情变换：

  1. 爬楼（爬上来再坐电梯下再爬上来）
  2. 跑步🏃‍♀️
  3. 踩椭圆机
  4. keep/b站视频跟练

# 团课

乐刻的一节团课大概 40min - 1h ~

  1. 白天不忙的情况下，中午或者傍晚吃饭时间去乐刻上一节团课

# 晚上回家

白天比较忙没有上团课的情况下，晚上回家跟着keep/b站做运动（半小时）

跟练视频参考：

  1. keep上帕梅拉老师的每周计划

  2. keep安排的训练计划（哑铃之类的无氧）

  3. 【学生党&上班族】减掉肚腩，比跑步更效率！| 无器械,零噪音跟练 (opens new window)

  4. 无跳动不伤膝盖超燃脂有氧运动，20分钟无工具低强度减肥操【周六野Zoey】 (opens new window)

  5. b站上一堆视频待补充...
